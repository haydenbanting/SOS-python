# SOS-python
The classic grid paper game, implemented in python. 

Dependencies:
1) NumPy
2) Pygame

Installation:
$ pip install numpy
$ pip install pygame
$ git clone https://github.com/haydenbanting/SOS-python.git

Running:
$ cd SOS-python
$ python main.py

Troubleshooting:
1) Pygame uses SDL, you may have to install/update SDL:
$ sudo apt-get install ia32-libs-sdl
